INTRODUCTION

INSTALLATION
------------

1. You'll need [Composer](http://getcomposer.org).
1. From withing this directory, run `composer install`
1. Copy `behat.local.yml.sample` to `behat.local.yml` and edit as needed
1. Configure `behat.yml` as needed

RUNNING TESTS
-------------

1. Run `bin/behat`
